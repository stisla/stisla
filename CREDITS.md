## Icons

### Solar Icon Set

- Author: 480 Design
- Source: https://github.com/480-Design/Solar-Icon-Set
- License: CC BY 4.0 — https://creativecommons.org/licenses/by/4.0/

Icons are used as-is. Color and size are applied via CSS; the source artwork
is not modified. Icons are baked into the shipped HTML as inline `<svg>` at
build time (`@iconify/utils` + `@iconify-json/solar`, dev dependencies only) —
there is no runtime icon library or network fetch in the distributed template.
