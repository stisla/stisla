Navs

A row of links for in-content navigation.

Basic

Wrap a list of .nav-item in .nav. Mark the current page with .active and unreachable links with .disabled. Link color stays the body text color across normal, hover, and active.

  • Active
  • Link
  • Another
  • Disabled
<ul class="nav">
  <li class="nav-item"><a class="nav-link active" aria-current="page" href="#">Active</a></li>
  <li class="nav-item"><a class="nav-link" href="#">Link</a></li>
  <li class="nav-item"><a class="nav-link" href="#">Another</a></li>
  <li class="nav-item"><a class="nav-link disabled" aria-disabled="true">Disabled</a></li>
</ul>

Vertical

Add .flex-column to stack the items. Pair with a breakpoint variant like .flex-sm-column to stack only below a width.

  • Active
  • Link
  • Another
  • Disabled
<ul class="nav flex-column" style="max-width: 12rem;">
  <li class="nav-item"><a class="nav-link active" aria-current="page" href="#">Active</a></li>
  <li class="nav-item"><a class="nav-link" href="#">Link</a></li>
  <li class="nav-item"><a class="nav-link" href="#">Another</a></li>
  <li class="nav-item"><a class="nav-link disabled" aria-disabled="true">Disabled</a></li>
</ul>

Tabs

Add .nav-tabs for a bordered card-style row. The active tab seats into the surface below; this variant is the static markup, jump to Tabs for the JS toggle that swaps content.

  • Active
  • Link
  • Another
  • Disabled
<ul class="nav nav-tabs">
  <li class="nav-item"><a class="nav-link active" aria-current="page" href="#">Active</a></li>
  <li class="nav-item"><a class="nav-link" href="#">Link</a></li>
  <li class="nav-item"><a class="nav-link" href="#">Another</a></li>
  <li class="nav-item"><a class="nav-link disabled" aria-disabled="true">Disabled</a></li>
</ul>

Pills

Add .nav-pills to give each row a filled background on hover and active. The chip uses the accent surface, matching the navbar and dropdown selection language.

  • Active
  • Link
  • Another
  • Disabled
<ul class="nav nav-pills">
  <li class="nav-item"><a class="nav-link active" aria-current="page" href="#">Active</a></li>
  <li class="nav-item"><a class="nav-link" href="#">Link</a></li>
  <li class="nav-item"><a class="nav-link" href="#">Another</a></li>
  <li class="nav-item"><a class="nav-link disabled" aria-disabled="true">Disabled</a></li>
</ul>

Use .nav-pills-alt for the alternative design.

  • Link
  • Another

Fill

Add .nav-fill so each item takes proportional space. Items keep their natural widths but share the available row.

  • Active
  • Much longer nav link
  • Link
  • Disabled
<ul class="nav nav-pills nav-fill">
  <li class="nav-item"><a class="nav-link active" aria-current="page" href="#">Active</a></li>
  <li class="nav-item"><a class="nav-link" href="#">Much longer nav link</a></li>
  <li class="nav-item"><a class="nav-link" href="#">Link</a></li>
  <li class="nav-item"><a class="nav-link disabled" aria-disabled="true">Disabled</a></li>
</ul>

With dropdown

Wrap a .dropdown in a .nav-item and use .nav-link.dropdown-toggle for the trigger, same pattern as the navbar.

  • Active
  • Dropdown
    • Action
    • Another action
    • Separated link
  • Link
  • Disabled
<ul class="nav nav-pills">
  <li class="nav-item"><a class="nav-link active" aria-current="page" href="#">Active</a></li>
  <li class="nav-item dropdown">
    <a class="nav-link dropdown-toggle" data-bs-toggle="dropdown" href="#" role="button" aria-expanded="false">Dropdown</a>
    <ul class="dropdown-menu">
      <li><a class="dropdown-item" href="#">Action</a></li>
      <li><a class="dropdown-item" href="#">Another action</a></li>
      <li><hr class="dropdown-divider"></li>
      <li><a class="dropdown-item" href="#">Separated link</a></li>
    </ul>
  </li>
  <li class="nav-item"><a class="nav-link" href="#">Link</a></li>
  <li class="nav-item"><a class="nav-link disabled" aria-disabled="true">Disabled</a></li>
</ul>
