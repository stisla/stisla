Tabs

A tab strip that swaps a panel of content on click.

Basic

Use buttons as the tab triggers and pair them with a .tab-content wrapper. Each trigger needs data-bs-toggle="tab" and a data-bs-target pointing at its pane. See Navs for the static row variants.

Home pane. Swap to another tab and this stays in the DOM.

Profile pane content.

Contact pane content.

<div class="w-100">
  <ul class="nav nav-tabs" id="tabsBasic" role="tablist">
    <li class="nav-item" role="presentation">
      <button class="nav-link active" id="tabsBasic-home-tab" data-bs-toggle="tab" data-bs-target="#tabsBasic-home" type="button" role="tab" aria-controls="tabsBasic-home" aria-selected="true">Home</button>
    </li>
    <li class="nav-item" role="presentation">
      <button class="nav-link" id="tabsBasic-profile-tab" data-bs-toggle="tab" data-bs-target="#tabsBasic-profile" type="button" role="tab" aria-controls="tabsBasic-profile" aria-selected="false">Profile</button>
    </li>
    <li class="nav-item" role="presentation">
      <button class="nav-link" id="tabsBasic-contact-tab" data-bs-toggle="tab" data-bs-target="#tabsBasic-contact" type="button" role="tab" aria-controls="tabsBasic-contact" aria-selected="false">Contact</button>
    </li>
  </ul>
  <div class="tab-content border border-top-0 rounded-bottom p-3" id="tabsBasicContent">
    <div class="tab-pane show active" id="tabsBasic-home" role="tabpanel" aria-labelledby="tabsBasic-home-tab" tabindex="0">
      <p class="mb-0">Home pane. Swap to another tab and this stays in the DOM.</p>
    </div>
    <div class="tab-pane" id="tabsBasic-profile" role="tabpanel" aria-labelledby="tabsBasic-profile-tab" tabindex="0">
      <p class="mb-0">Profile pane content.</p>
    </div>
    <div class="tab-pane" id="tabsBasic-contact" role="tabpanel" aria-labelledby="tabsBasic-contact-tab" tabindex="0">
      <p class="mb-0">Contact pane content.</p>
    </div>
  </div>
</div>

Vertical pills

Combine .flex-column with .nav-pills and seat the pane container beside the strip in a grid row. Add aria-orientation="vertical" on the tab list for assistive tech.
