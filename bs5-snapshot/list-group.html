List group

A flexible stack of rows on a shared rounded surface. Use it for navigation, settings, contact lists, payment methods, or any vertical run of similar items.

Basic

A .list-group with plain .list-group-item rows. Each row sits on the surface tier and shares borders with its neighbors.

  • Cras justo odio
  • Dapibus ac facilisis in
  • Morbi leo risus
  • Porta ac consectetur ac
  • Vestibulum at eros
<ul class="list-group" style="max-width: 24rem;">
  <li class="list-group-item">Cras justo odio</li>
  <li class="list-group-item">Dapibus ac facilisis in</li>
  <li class="list-group-item">Morbi leo risus</li>
  <li class="list-group-item">Porta ac consectetur ac</li>
  <li class="list-group-item">Vestibulum at eros</li>
</ul>

Active and disabled

Mark the current item with .active and unreachable items with .disabled. Active takes the accent surface; disabled greys out and stops pointer events.

  • An active item
  • A second item
  • A third item
  • A disabled item
<ul class="list-group" style="max-width: 24rem;">
  <li class="list-group-item active" aria-current="true">An active item</li>
  <li class="list-group-item">A second item</li>
  <li class="list-group-item">A third item</li>
  <li class="list-group-item disabled" aria-disabled="true">A disabled item</li>
</ul>

Links and buttons

Swap <li> for <a> or <button> and add .list-group-item-action to make rows interactive. Hover and focus light up; .active sticks.

The current link A second link A third link A disabled link 
<div class="list-group" style="max-width: 24rem;">
  <a href="#" class="list-group-item list-group-item-action active" aria-current="true">The current link</a>
  <a href="#" class="list-group-item list-group-item-action">A second link</a>
  <a href="#" class="list-group-item list-group-item-action">A third link</a>
  <a href="#" class="list-group-item list-group-item-action disabled" aria-disabled="true">A disabled link</a>
</div>

Flush

Add .list-group-flush to drop the wrapper border and radius. Rows sit edge-to-edge with a single divider between them — useful inside cards or panels.

  • An item
  • A second item
  • A third item
  • A fourth item
<ul class="list-group list-group-flush" style="max-width: 24rem;">
  <li class="list-group-item">An item</li>
  <li class="list-group-item">A second item</li>
  <li class="list-group-item">A third item</li>
  <li class="list-group-item">A fourth item</li>
</ul>

Numbered

Drop .list-group-numbered on the wrapper and use an <ol>. Bootstrap stamps the counter; nest another numbered list and the counter cascades.

  1. Pour a flat white
  2. Open the laptop
  3. Ship something
<ol class="list-group list-group-numbered" style="max-width: 24rem;">
  <li class="list-group-item">Pour a flat white</li>
  <li class="list-group-item">Open the laptop</li>
  <li class="list-group-item">Ship something</li>
</ol>

Horizontal

Use .list-group-horizontal to lay rows side by side instead of stacked. Add a breakpoint suffix (-md, -lg, …) to switch at a given width.

  • First
  • Second
  • Third
<ul class="list-group list-group-horizontal">
  <li class="list-group-item">First</li>
  <li class="list-group-item">Second</li>
  <li class="list-group-item">Third</li>
</ul>

With badge

Push a count or status to the trailing edge with d-flex justify-content-between and a .badge. Reads as a counter row.

  • Inbox 14
  • Drafts 2
  • Spam 0 
<ul class="list-group" style="max-width: 24rem; width: 12rem">
  <li class="list-group-item d-flex justify-content-between align-items-center">
    Inbox
    <span class="badge text-bg-primary">14</span>
  </li>
  <li class="list-group-item d-flex justify-content-between align-items-center">
    Drafts
    <span class="badge badge-light-secondary">2</span>
  </li>
  <li class="list-group-item d-flex justify-content-between align-items-center">
    Spam
    <span class="badge badge-light-secondary">0</span>
  </li>
</ul>

Contextual variants

Tint a row with .list-group-item-{state} for status messaging. Same five semantic states as alerts.

  • A neutral item
  • A primary item
  • A success item
  • An info item
  • A warning item
  • A danger item
<ul class="list-group" style="max-width: 24rem;">
  <li class="list-group-item">A neutral item</li>
  <li class="list-group-item list-group-item-primary">A primary item</li>
  <li class="list-group-item list-group-item-success">A success item</li>
  <li class="list-group-item list-group-item-info">An info item</li>
  <li class="list-group-item list-group-item-warning">A warning item</li>
  <li class="list-group-item list-group-item-danger">A danger item</li>
</ul>

Settings

Use .list-group-flush inside a .card for a settings panel. Each row pairs a label and helper text with the matching control — switch, select, or input — at the trailing edge.

Notifications
  • A weekly summary of activity across your projects.
  • Get a ping on this device when someone mentions you.
  • Used across the app and outgoing emails.
  • Shown on your profile and in mentions.
  • Delete workspace
    This permanently removes everything. Cannot be undone.
