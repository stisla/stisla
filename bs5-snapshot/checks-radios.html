Checks & radios

Checkboxes, radios, and switches for choosing one or more options.

Checkbox

Wrap an <input type="checkbox"> in .form-check and pair it with a .form-check-label tied via for/id.

<div class="w-100" style="max-width: 24rem;">
  <div class="form-check">
    <input class="form-check-input" type="checkbox" id="defaultCheck" />
    <label class="form-check-label" for="defaultCheck">Default checkbox</label>
  </div>
  <div class="form-check">
    <input class="form-check-input" type="checkbox" id="checkedCheck" checked />
    <label class="form-check-label" for="checkedCheck">Checked by default</label>
  </div>
</div>

Radio

Group radios by giving them a shared name. The browser handles single-selection within the group.

<div class="w-100" style="max-width: 24rem;">
  <div class="form-check">
    <input class="form-check-input" type="radio" name="defaultRadio" id="defaultRadio1" checked />
    <label class="form-check-label" for="defaultRadio1">First option</label>
  </div>
  <div class="form-check">
    <input class="form-check-input" type="radio" name="defaultRadio" id="defaultRadio2" />
    <label class="form-check-label" for="defaultRadio2">Second option</label>
  </div>
</div>

Inline

Add .form-check-inline to line items up on the same row.

<div>
  <div class="form-check form-check-inline">
    <input class="form-check-input" type="checkbox" id="inlineCheck1" />
    <label class="form-check-label" for="inlineCheck1">One</label>
  </div>
  <div class="form-check form-check-inline">
    <input class="form-check-input" type="checkbox" id="inlineCheck2" />
    <label class="form-check-label" for="inlineCheck2">Two</label>
  </div>
  <div class="form-check form-check-inline">
    <input class="form-check-input" type="checkbox" id="inlineCheck3" />
    <label class="form-check-label" for="inlineCheck3">Three</label>
  </div>
</div>

Switch

Add .form-switch alongside .form-check to render a checkbox as a toggle.

<div class="w-100" style="max-width: 24rem;">
  <div class="form-check form-switch">
    <input class="form-check-input" type="checkbox" role="switch" id="defaultSwitch" />
    <label class="form-check-label" for="defaultSwitch">Notifications</label>
  </div>
  <div class="form-check form-switch">
    <input class="form-check-input" type="checkbox" role="switch" id="checkedSwitch" checked />
    <label class="form-check-label" for="checkedSwitch">Auto-update</label>
  </div>
</div>

Large switch

Add .form-switch-lg for a 44×24 variant. Better suited to standalone settings rows where the switch is the row's primary affordance.

<div class="w-100" style="max-width: 24rem;">
  <div class="form-check form-switch form-switch-lg">
    <input class="form-check-input" type="checkbox" role="switch" id="lgSwitch" />
    <label class="form-check-label" for="lgSwitch">Notifications</label>
  </div>
  <div class="form-check form-switch form-switch-lg">
    <input class="form-check-input" type="checkbox" role="switch" id="lgSwitchOn" checked />
    <label class="form-check-label" for="lgSwitchOn">Auto-update</label>
  </div>
</div>

Settings row

Drop the switch into a flex row to put the label on the left and the affordance on the right. The .form-check wrapper stays single-purpose; row layout lives on the outer container.

<div class="w-100" style="max-width: 24rem;">
  <div class="d-flex align-items-center justify-content-between py-2">
    <label class="form-check-label" for="settingEmail">Email notifications</label>
    <div class="form-check form-switch form-switch-lg p-0 m-0">
      <input class="form-check-input m-0" type="checkbox" role="switch" id="settingEmail" checked />
    </div>
  </div>
  <div class="d-flex align-items-center justify-content-between py-2">
    <label class="form-check-label" for="settingPush">Push notifications</label>
    <div class="form-check form-switch form-switch-lg p-0 m-0">
      <input class="form-check-input m-0" type="checkbox" role="switch" id="settingPush" />
    </div>
  </div>
</div>

Indeterminate

The indeterminate state is set from script — there's no HTML attribute for it. Useful as a parent of a partially-selected group.

<div class="w-100" style="max-width: 24rem;">
  <div class="form-check">
    <input class="form-check-input" type="checkbox" id="indeterminateCheck" />
    <label class="form-check-label" for="indeterminateCheck">Select all</label>
  </div>
  <script>
    document.getElementById('indeterminateCheck').indeterminate = true;
  </script>
</div>

Reverse

Add .form-check-reverse to flip the label to the start and the input to the end.

<div class="w-100" style="max-width: 24rem;">
  <div class="form-check form-check-reverse">
    <input class="form-check-input" type="checkbox" id="reverseCheck" />
    <label class="form-check-label" for="reverseCheck">Reversed checkbox</label>
  </div>
  <div class="form-check form-check-reverse form-switch">
    <input class="form-check-input" type="checkbox" role="switch" id="reverseSwitch" checked />
    <label class="form-check-label" for="reverseSwitch">Reversed switch</label>
  </div>
</div>

Disabled

Add disabled to dim the input and its label, and block interaction.

<div class="w-100" style="max-width: 24rem;">
  <div class="form-check">
    <input class="form-check-input" type="checkbox" id="disabledCheck" disabled />
    <label class="form-check-label" for="disabledCheck">Disabled checkbox</label>
  </div>
  <div class="form-check">
    <input class="form-check-input" type="checkbox" id="disabledCheckedCheck" checked disabled />
    <label class="form-check-label" for="disabledCheckedCheck">Disabled, checked</label>
  </div>
  <div class="form-check form-switch">
    <input class="form-check-input" type="checkbox" role="switch" id="disabledSwitch" checked disabled />
    <label class="form-check-label" for="disabledSwitch">Disabled switch</label>
  </div>
</div>

Toggle buttons

Pair .btn-check with a sibling .btn label to render a checkbox or radio as a button. The input stays visually hidden but reachable by keyboard.

<div class="d-flex flex-column gap-3">
  <div>
    <input type="checkbox" class="btn-check" id="toggleCheck" autocomplete="off" />
    <label class="btn btn-outline-primary" for="toggleCheck">Single toggle</label>
  </div>
  <div>
    <input type="radio" class="btn-check" name="toggleRadio" id="toggleRadio1" autocomplete="off" checked />
    <label class="btn btn-outline-primary" for="toggleRadio1">Day</label>
    <input type="radio" class="btn-check" name="toggleRadio" id="toggleRadio2" autocomplete="off" />
    <label class="btn btn-outline-primary" for="toggleRadio2">Week</label>
    <input type="radio" class="btn-check" name="toggleRadio" id="toggleRadio3" autocomplete="off" />
    <label class="btn btn-outline-primary" for="toggleRadio3">Month</label>
  </div>
</div>

Without labels

Drop the .form-check wrapper to use a bare .form-check-input. Always pair with an aria-label for accessibility.

<div class="d-flex align-items-center gap-3">
  <input class="form-check-input" type="checkbox" aria-label="Bare checkbox" />
  <input class="form-check-input" type="radio" aria-label="Bare radio" checked />
  <input class="form-check-input" type="checkbox" role="switch" aria-label="Bare switch" checked />
</div>
