Cards

A flexible content container with body, optional header / footer, image, and list-group regions.

Basic

A card with just .card-body.

This is some text within a card body. 
<div class="card" style="width: 18rem;">
  <div class="card-body">
    This is some text within a card body.
  </div>
</div>

Titles, subtitle, text, links

.card-title is pinned to 16px regardless of which <h*> tag you pick. .card-subtitle picks up --bs-secondary-color so the muted treatment is baked in: no .text-body-secondary utility needed.

Sizing

Cards have no built-in width. Size them with inline styles (style="width: 18rem"), utility classes (.w-75, .w-50), or grid columns.

style="width: 18rem"

Inline width.

.w-75

Utility width.

.w-50

Utility width.

<div>
  <div class="card mb-3" style="width: 18rem;">
    <div class="card-body">
      <h5 class="card-title">style="width: 18rem"</h5>
      <p class="card-text mb-0">Inline width.</p>
    </div>
  </div>
  <div class="card w-75 mb-3">
    <div class="card-body">
      <h5 class="card-title">.w-75</h5>
      <p class="card-text mb-0">Utility width.</p>
    </div>
  </div>
  <div class="card w-50">
    <div class="card-body">
      <h5 class="card-title">.w-50</h5>
      <p class="card-text mb-0">Utility width.</p>
    </div>
  </div>
</div>

Or use the grid for responsive sizing.
