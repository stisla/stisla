Navbar

A top bar with a brand mark and a row of nav links.

Basic

Wrap a .navbar-brand and a .navbar-nav in .navbar.navbar-expand-lg. The .navbar-toggler + .navbar-collapse pair folds the menu below the expand breakpoint; resize the preview to a small width to see it fold.

Acme
  • Home
  • Features
  • Pricing
  • About
<nav class="navbar navbar-expand-lg border rounded w-100">
  <a class="navbar-brand" href="#">Acme</a>
  <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarBasic" aria-controls="navbarBasic" aria-expanded="false" aria-label="Toggle navigation">
    <span class="navbar-toggler-icon"></span>
  </button>
  <div class="collapse navbar-collapse" id="navbarBasic">
    <ul class="navbar-nav">
      <li class="nav-item"><a class="nav-link active" aria-current="page" href="#">Home</a></li>
      <li class="nav-item"><a class="nav-link" href="#">Features</a></li>
      <li class="nav-item"><a class="nav-link" href="#">Pricing</a></li>
      <li class="nav-item"><a class="nav-link" href="#">About</a></li>
    </ul>
  </div>
</nav>

With dropdown

Drop a .dropdown into the nav list. The toggle uses .nav-link.dropdown-toggle so it shares the same ghost chip as the siblings.

Acme
  • Home
  • Features
  • Resources
    • Docs
    • Changelog
    • Support
<nav class="navbar navbar-expand-lg border rounded w-100">
  <a class="navbar-brand" href="#">Acme</a>
  <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarDropdown" aria-controls="navbarDropdown" aria-expanded="false" aria-label="Toggle navigation">
    <span class="navbar-toggler-icon"></span>
  </button>
  <div class="collapse navbar-collapse" id="navbarDropdown">
    <ul class="navbar-nav">
      <li class="nav-item"><a class="nav-link active" aria-current="page" href="#">Home</a></li>
      <li class="nav-item"><a class="nav-link" href="#">Features</a></li>
      <li class="nav-item dropdown">
        <a class="nav-link dropdown-toggle" href="#" role="button" data-bs-toggle="dropdown" aria-expanded="false">
          Resources
        </a>
        <ul class="dropdown-menu">
          <li><a class="dropdown-item" href="#">Docs</a></li>
          <li><a class="dropdown-item" href="#">Changelog</a></li>
          <li><hr class="dropdown-divider"></li>
          <li><a class="dropdown-item" href="#">Support</a></li>
        </ul>
      </li>
    </ul>
  </div>
</nav>

Active and disabled

Mark the current page with .active and unreachable links with .disabled. Active takes the emphasis color; disabled greys out and stops pointer events.

Acme
  • Overview
  • Reports
  • Settings
  • Billing
<nav class="navbar navbar-expand-lg border rounded w-100">
  <a class="navbar-brand" href="#">Acme</a>
  <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarStates" aria-controls="navbarStates" aria-expanded="false" aria-label="Toggle navigation">
    <span class="navbar-toggler-icon"></span>
  </button>
  <div class="collapse navbar-collapse" id="navbarStates">
    <ul class="navbar-nav">
      <li class="nav-item"><a class="nav-link" href="#">Overview</a></li>
      <li class="nav-item"><a class="nav-link active" aria-current="page" href="#">Reports</a></li>
      <li class="nav-item"><a class="nav-link" href="#">Settings</a></li>
      <li class="nav-item"><a class="nav-link disabled" aria-disabled="true">Billing</a></li>
    </ul>
  </div>
</nav>

With search

Drop a <form> after the nav list and use .me-auto on the list to push the form to the trailing edge.

Acme
  • Home
  • Features
<nav class="navbar navbar-expand-lg border rounded w-100">
  <a class="navbar-brand" href="#">Acme</a>
  <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarSearch" aria-controls="navbarSearch" aria-expanded="false" aria-label="Toggle navigation">
    <span class="navbar-toggler-icon"></span>
  </button>
  <div class="collapse navbar-collapse" id="navbarSearch">
    <ul class="navbar-nav me-auto">
      <li class="nav-item"><a class="nav-link active" aria-current="page" href="#">Home</a></li>
      <li class="nav-item"><a class="nav-link" href="#">Features</a></li>
    </ul>
    <form class="d-flex gap-2 ms-0 ms-lg-4" role="search">
      <input class="form-control" type="search" placeholder="Search" aria-label="Search">
      <button class="btn btn-primary" type="submit">Search</button>
    </form>
  </div>
</nav>

With text

Use .navbar-text for non-interactive copy like a status pill or a workspace name on the trailing edge.

Acme
All systems normal 
<nav class="navbar navbar-expand-lg border rounded w-100">
  <a class="navbar-brand" href="#">Acme</a>
  <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarText" aria-controls="navbarText" aria-expanded="false" aria-label="Toggle navigation">
    <span class="navbar-toggler-icon"></span>
  </button>
  <div class="collapse navbar-collapse" id="navbarText">
    <span class="navbar-text d-inline-flex align-items-center ms-auto">
      <span class="bg-success rounded-circle d-inline-block me-2" style="width: 6px; height: 6px;"></span>
      All systems normal
    </span>
  </div>
</nav>

Container

Drop a .container or .container-fluid as the bar's direct child to align its contents with a matching container in the page body. The bar still spans full width; only the inner row clamps to the container's gutter.

Acme
  • Home
  • Features
  • Pricing
<nav class="navbar navbar-expand-lg border rounded w-100">
  <div class="container">
    <a class="navbar-brand" href="#">Acme</a>
    <button class="navbar-toggler" type="button" data-bs-toggle="collapse" data-bs-target="#navbarContainer" aria-controls="navbarContainer" aria-expanded="false" aria-label="Toggle navigation">
      <span class="navbar-toggler-icon"></span>
    </button>
    <div class="collapse navbar-collapse" id="navbarContainer">
      <ul class="navbar-nav">
        <li class="nav-item"><a class="nav-link active" aria-current="page" href="#">Home</a></li>
        <li class="nav-item"><a class="nav-link" href="#">Features</a></li>
        <li class="nav-item"><a class="nav-link" href="#">Pricing</a></li>
      </ul>
    </div>
  </div>
</nav>
