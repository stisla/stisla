Badge

A compact label for counts, statuses, and tags.

Variants

Pair .badge with a .text-bg-{state} utility for the solid filled chip.

Primary Secondary Success Danger Warning Info 
<div class="d-flex flex-wrap gap-2">
  <span class="badge text-bg-primary">Primary</span>
  <span class="badge text-bg-secondary">Secondary</span>
  <span class="badge text-bg-success">Success</span>
  <span class="badge text-bg-danger">Danger</span>
  <span class="badge text-bg-warning">Warning</span>
  <span class="badge text-bg-info">Info</span>
</div>

Light

Use .badge-light-{state} for a soft tinted chip.

Primary Secondary Success Danger Warning Info 
<div class="d-flex flex-wrap gap-2">
  <span class="badge badge-light-primary">Primary</span>
  <span class="badge badge-light-secondary">Secondary</span>
  <span class="badge badge-light-success">Success</span>
  <span class="badge badge-light-danger">Danger</span>
  <span class="badge badge-light-warning">Warning</span>
  <span class="badge badge-light-info">Info</span>
</div>

Outline

Use .badge-outline-{state} for a soft tinted chip with border.

Primary Secondary Success Danger Warning Info 
<div class="d-flex flex-wrap gap-2">
  <span class="badge badge-outline-primary">Primary</span>
  <span class="badge badge-outline-secondary">Secondary</span>
  <span class="badge badge-outline-success">Success</span>
  <span class="badge badge-outline-danger">Danger</span>
  <span class="badge badge-outline-warning">Warning</span>
  <span class="badge badge-outline-info">Info</span>
</div>

With icon

Drop a direct <svg> or <i> child. The icon picks up the text color and scales to the badge's font size.

Active Pending Failed Featured 
<div class="d-flex flex-wrap gap-2">
  <span class="badge badge-light-success">
    <i data-lucide="check"></i>
    Active
  </span>
  <span class="badge badge-light-warning">
    <i data-lucide="clock"></i>
    Pending
  </span>
  <span class="badge badge-light-danger">
    <i data-lucide="x"></i>
    Failed
  </span>
  <span class="badge text-bg-primary">
    <i data-lucide="star"></i>
    Featured
  </span>
</div>

Loading

Slot a Bootstrap .spinner-border in to signal in-flight work. It inherits the badge's text color and shrinks to the same metric as an icon.

Syncing Loading Uploading 
<div class="d-flex flex-wrap gap-2">
  <span class="badge badge-light-primary">
    <span class="spinner-border" role="status" aria-hidden="true"></span>
    Syncing
  </span>
  <span class="badge badge-light-secondary">
    <span class="spinner-border" role="status" aria-hidden="true"></span>
    Loading
  </span>
  <span class="badge text-bg-info">
    <span class="spinner-border" role="status" aria-hidden="true"></span>
    Uploading
  </span>
</div>

Pill

Add .rounded-pill for a fully rounded shape.

Primary Success Warning Info 
<div class="d-flex flex-wrap gap-2">
  <span class="badge rounded-pill text-bg-primary">Primary</span>
  <span class="badge rounded-pill text-bg-success">Success</span>
  <span class="badge rounded-pill badge-light-warning">Warning</span>
  <span class="badge rounded-pill badge-light-info">Info</span>
</div>

In buttons

Drop a badge inside a button for a count or status. Combine with a .btn-icon for a notification dot.

<div class="d-flex flex-wrap gap-2 align-items-center">
  <button type="button" class="btn btn-primary">
    Inbox
    <span class="badge text-bg-light">4</span>
  </button>
  <button type="button" class="btn btn-outline-secondary">
    Notifications
    <span class="badge text-bg-danger">12</span>
  </button>
</div>
