App shell

A layout that positions Stisla's .sidebar next to a main content column.

Basic

.app-shell wraps a .sidebar and an .app-shell-main. The shell sets the sidebar's width via --bs-app-shell-sidebar-width (default 16rem), and main fills the rest with flex-grow: 1.

Sticky sidebar

The shell defaults to min-height: 100vh. When the page is taller than the viewport, the window scrolls and the sidebar scrolls with it. To pin the sidebar to the viewport top, add four inline styles to .sidebar.

<div class="app-shell">
  <aside class="sidebar" style="
    position: sticky;
    top: 0;
    height: 100vh;
    align-self: flex-start;
  ">
    <!-- … -->
  </aside>
  <main class="app-shell-main">
    <!-- long content here; the window scrolls, the sidebar stays put -->
  </main>
</div>

For a Gmail-style locked shell (no window scroll, main is the scroll container), use height: 100vh; overflow: hidden on the shell and overflow: auto on main. Both patterns are opt-in via inline style, and the shell stays unopinionated.

Desktop collapse (rail)

Add .is-sidebar-collapsed to .app-shell to swap the sidebar to its narrow rail width (--bs-app-shell-sidebar-width-collapsed, default 4.25rem). Pair it with .is-collapsed on the sidebar itself for the rail visuals. Labels fade out, icons stay centered. Any button can wire the toggle with data-app-shell-toggle="collapse".

Mobile drawer

Below the lg breakpoint, the sidebar leaves the flex flow and sits fixed off-screen. Setting .is-sidebar-visible on the shell slides it in and shows a backdrop. Any button can flip the class with data-app-shell-toggle="visibility".

Theming

Component vars override at any .app-shell instance. The shell's outer wrapper and the main column each have their own bg + text vars, and the sidebar carries its own --bs-sidebar-bg separately.

<div class="app-shell" style="
  --bs-app-shell-bg: #f5f7fa;
  --bs-app-shell-main-bg: #ffffff;
">
  <aside class="sidebar" style="--bs-sidebar-bg: transparent;">
    <!-- … -->
  </aside>
  <main class="app-shell-main">
    <!-- … -->
  </main>
</div>

Available shell vars: --bs-app-shell-bg, --bs-app-shell-color, --bs-app-shell-main-bg, --bs-app-shell-main-color, --bs-app-shell-sidebar-width, --bs-app-shell-sidebar-width-collapsed, --bs-app-shell-backdrop-bg, --bs-app-shell-transition-duration.
