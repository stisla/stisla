Carousel

A slideshow that cycles through any content.

Slides only

The bare component cycles through .carousel-items. Mark one as .active to set the starting slide.

With controls

Add .carousel-control-prev and .carousel-control-next with data-bs-target pointing at the carousel. The pair carries .btn .btn-outline-secondary so the chrome tracks the outline-secondary variant, then the component file pins it as a 1:1 circle with a smaller chevron.
