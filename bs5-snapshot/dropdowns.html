Dropdowns

A floating menu of links or actions surfaced from a trigger.

Basic

Pair a .dropdown-toggle with a .dropdown-menu, both inside .dropdown. Bootstrap's data API handles open and close.

  • Profile
  • Settings
  • Sign out
<div class="dropdown">
  <button class="btn btn-outline-secondary dropdown-toggle" type="button" data-bs-toggle="dropdown" aria-expanded="false">
    Actions
  </button>
  <ul class="dropdown-menu">
    <li><a class="dropdown-item" href="#">Profile</a></li>
    <li><a class="dropdown-item" href="#">Settings</a></li>
    <li><a class="dropdown-item" href="#">Sign out</a></li>
  </ul>
</div>

With icons

Drop a <svg> or <i> as the first child of each item. The icon sits at 16px and inherits the row's color on hover.

  • Profile
  • Settings
  • Notifications
  • Sign out 
<div class="dropdown">
  <button class="btn btn-outline-secondary dropdown-toggle" type="button" data-bs-toggle="dropdown" aria-expanded="false">
    Account
  </button>
  <ul class="dropdown-menu">
    <li>
      <a class="dropdown-item" href="#">
        <i data-lucide="user"></i>
        <span>Profile</span>
      </a>
    </li>
    <li>
      <a class="dropdown-item" href="#">
        <i data-lucide="settings"></i>
        <span>Settings</span>
      </a>
    </li>
    <li>
      <a class="dropdown-item" href="#">
        <i data-lucide="bell"></i>
        <span>Notifications</span>
      </a>
    </li>
    <li><hr class="dropdown-divider"></li>
    <li>
      <a class="dropdown-item" href="#">
        <i data-lucide="log-out"></i>
        <span>Sign out</span>
      </a>
    </li>
  </ul>
</div>

Headers and dividers

Use .dropdown-header to label a section and .dropdown-divider to separate groups.

  • Account
  • Profile
  • Billing
  • Workspace
  • Members
  • Settings
<div class="dropdown">
  <button class="btn btn-outline-secondary dropdown-toggle" type="button" data-bs-toggle="dropdown" aria-expanded="false">
    Menu
  </button>
  <ul class="dropdown-menu">
    <li><h6 class="dropdown-header">Account</h6></li>
    <li><a class="dropdown-item" href="#">Profile</a></li>
    <li><a class="dropdown-item" href="#">Billing</a></li>
    <li><hr class="dropdown-divider"></li>
    <li><h6 class="dropdown-header">Workspace</h6></li>
    <li><a class="dropdown-item" href="#">Members</a></li>
    <li><a class="dropdown-item" href="#">Settings</a></li>
  </ul>
</div>

Active and disabled

Mark the current row with .active and unreachable rows with .disabled. Active takes the accent surface; disabled greys out and stops pointer events.

  • Newest first
  • Oldest first
  • Alphabetical
  • By owner (Pro)
<div class="dropdown">
  <button class="btn btn-outline-secondary dropdown-toggle" type="button" data-bs-toggle="dropdown" aria-expanded="false">
    Sort by
  </button>
  <ul class="dropdown-menu">
    <li><a class="dropdown-item active" href="#" aria-current="true">Newest first</a></li>
    <li><a class="dropdown-item" href="#">Oldest first</a></li>
    <li><a class="dropdown-item" href="#">Alphabetical</a></li>
    <li><a class="dropdown-item disabled" href="#" aria-disabled="true">By owner (Pro)</a></li>
  </ul>
</div>

Split button

Pair a primary action with a caret-only toggle. Wrap both in .btn-group so they share rounding.

  • Save and continue
  • Save as draft
  • Discard changes
<div class="btn-group">
  <button type="button" class="btn btn-primary">Save</button>
  <button type="button" class="btn btn-primary dropdown-toggle dropdown-toggle-split" data-bs-toggle="dropdown" aria-expanded="false">
    <span class="visually-hidden">Toggle dropdown</span>
  </button>
  <ul class="dropdown-menu">
    <li><a class="dropdown-item" href="#">Save and continue</a></li>
    <li><a class="dropdown-item" href="#">Save as draft</a></li>
    <li><hr class="dropdown-divider"></li>
    <li><a class="dropdown-item" href="#">Discard changes</a></li>
  </ul>
</div>

Drop direction

Swap .dropdown for .dropup, .dropend, or .dropstart to open the menu above or to the side of the trigger.

  • Action
  • Another action
  • Action
  • Another action
  • Action
  • Another action

  <div class="dropup">
    <button class="btn btn-outline-secondary dropdown-toggle" type="button" data-bs-toggle="dropdown" aria-expanded="false">
      Dropup
    </button>
    <ul class="dropdown-menu">
      <li><a class="dropdown-item" href="#">Action</a></li>

  </div>

Form

Drop a <form> straight into the menu. Pad with utilities and set a min-width to give the fields room.

<div class="dropdown">
  <button class="btn btn-outline-secondary dropdown-toggle" type="button" data-bs-toggle="dropdown" data-bs-auto-close="outside" aria-expanded="false">
    Sign in
  </button>
  <form class="dropdown-menu p-3" style="min-width: 18rem">
    <div class="mb-3">
      <label for="dropdown-form-email" class="form-label">Email</label>
      <input type="email" class="form-control" id="dropdown-form-email" placeholder="you@example.com">
    </div>
    <div class="mb-3">
      <label for="dropdown-form-password" class="form-label">Password</label>
      <input type="password" class="form-control" id="dropdown-form-password">
    </div>
    <div class="form-check mb-3">
      <input type="checkbox" class="form-check-input" id="dropdown-form-remember">
      <label class="form-check-label" for="dropdown-form-remember">Remember me</label>
    </div>
    <button type="submit" class="btn btn-primary w-100">Sign in</button>
  </form>
</div>
