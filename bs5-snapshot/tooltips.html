Tooltips

A short floating label anchored to a trigger.

Basic

Set data-bs-toggle="tooltip" and pass the label via data-bs-title. The tooltip initializes from a one-line scanner in site.js.

<button type="button" class="btn btn-primary" data-bs-toggle="tooltip" data-bs-title="Saved 2 minutes ago">
  Hover for status
</button>

Inverted by theme

The tooltip surface flips with the active theme. In light mode the chip reads as a near-black pill on a white page; in dark mode it inverts to near-white on the dark surface. The flip is automatic via --bs-emphasis-color and --bs-body-bg, no per-mode override needed.

<div class="d-flex flex-wrap gap-2">
  <button type="button" class="btn btn-outline-secondary" data-bs-toggle="tooltip" data-bs-title="Inverts with the page">
    Hover me
  </button>
  <button type="button" class="btn btn-outline-secondary" data-bs-toggle="tooltip" data-bs-title="Try the theme toggle in the top nav">
    Then toggle theme
  </button>
</div>

Placements

Use data-bs-placement to anchor the chip on a specific side of the trigger.

<div class="d-flex flex-wrap gap-2">
  <button type="button" class="btn btn-outline-secondary" data-bs-toggle="tooltip" data-bs-placement="top" data-bs-title="Anchored above">
    Top
  </button>
  <button type="button" class="btn btn-outline-secondary" data-bs-toggle="tooltip" data-bs-placement="right" data-bs-title="Anchored right">
    Right
  </button>
  <button type="button" class="btn btn-outline-secondary" data-bs-toggle="tooltip" data-bs-placement="bottom" data-bs-title="Anchored below">
    Bottom
  </button>
  <button type="button" class="btn btn-outline-secondary" data-bs-toggle="tooltip" data-bs-placement="left" data-bs-title="Anchored left">
    Left
  </button>
</div>

On a link

Tooltips also attach to inline elements. Useful for glossing acronyms or pointing at a destination.

The build runs on CI every push and posts a summary to the release channel. 

<p class="mb-0">
  The build runs on
  <a href="#" data-bs-toggle="tooltip" data-bs-title="GitHub Actions">CI</a>
  every push and posts a summary to
  <a href="#" data-bs-toggle="tooltip" data-bs-title="#releases on Slack">the release channel</a>.
</p>

HTML content

Pass data-bs-html="true" to render the title as HTML instead of escaping it. Keep the markup short, tooltips aren't popovers.

<button type="button" class="btn btn-outline-secondary" data-bs-toggle="tooltip" data-bs-html="true" data-bs-title="Press <kbd>⌘</kbd>+<kbd>K</kbd> to search">
  Open command palette
</button>

Long content

Content past --bs-tooltip-max-width (240px) wraps to a second line. Two lines is the practical ceiling, anything longer belongs in a popover.

<button type="button" class="btn btn-outline-secondary" data-bs-toggle="tooltip" data-bs-title="Only workspace owners can change billing details and downgrade the plan">
  Hover for the rule
</button>

Disabled trigger

Disabled buttons don't fire pointer events, so wrap them in a focusable span and put the tooltip attributes on the wrapper.

<span class="d-inline-block" tabindex="0" data-bs-toggle="tooltip" data-bs-title="Upgrade to enable exports">
  <button type="button" class="btn btn-primary" disabled style="pointer-events: none;">Export CSV</button>
</span>
