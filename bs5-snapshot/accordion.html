Accordion

A stack of collapsible panels. Each item is a self-contained rounded chip with a 4px inset from the wrapper edge; the flush variant strips the chip styling for an edge-to-edge list.

Basic

Default behavior: opening one item collapses the others via data-bs-parent. The first item is open on load.

This is the first item's accordion body. It shows by default. The chip is fully rounded; the body fills the chip's bottom corners.

Active items use a --bs-primary 15% alpha tint with full primary text and chevron.

Each chip carries its own border and 16px radius; BS5's stacked-block first/last special-casing is overridden so every item reads the same.

Always open

Drop data-bs-parent on each .accordion-collapse to let multiple items stay open at once.

This item stays open even when others are toggled.

Toggling this item does not collapse the others.

Open as many as you like.

Flush

Add .accordion-flush to drop the chip styling: no wrapper padding, no per-item radius, and a single divider between rows. The active state stays plain — no primary tint.

Liverpool Football Club is the biggest football club in the world.

Across all competitions, well over 60 trophies — domestic and continental.

Probably not — but you'll never walk alone.
<div class="accordion accordion-flush w-100" id="accordionFlushExample">
  <div class="accordion-item">
    <h2 class="accordion-header">
      <button class="accordion-button" type="button" data-bs-toggle="collapse" data-bs-target="#flush-collapseOne" aria-expanded="true" aria-controls="flush-collapseOne">
        What is Liverpool Football Club?
      </button>
    </h2>
