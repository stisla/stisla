Placeholders

A skeleton stand-in for content that hasn't loaded yet.

Example

The same card in two states. The loading card mirrors the real card's layout so nothing jumps when content arrives. Wrap the skeleton tree in .text-body-secondary so the bars sit on a muted gray instead of the high-contrast body color.

Quarterly review

Wraps up the metrics, customer wins, and what's queued for next quarter.

Open report 

<div class="row g-3 w-100">
  <div class="col-md-6">
    <div class="card">
      <div class="card-body">
        <h5 class="card-title">Quarterly review</h5>
        <p class="card-text">Wraps up the metrics, customer wins, and what's queued for next quarter.</p>
        <a href="#" class="btn btn-primary">Open report</a>
      </div>
    </div>
  </div>
  <div class="col-md-6">
    <div class="card" aria-hidden="true">
      <div class="card-body text-body-secondary">
        <h5 class="card-title placeholder-glow">
          <span class="placeholder col-6"></span>
        </h5>
        <p class="card-text placeholder-glow">
          <span class="placeholder col-7"></span>
          <span class="placeholder col-4"></span>
          <span class="placeholder col-4"></span>
          <span class="placeholder col-6"></span>
          <span class="placeholder col-8"></span>
        </p>
        <a class="btn btn-primary disabled placeholder col-4" aria-disabled="true"></a>
      </div>
    </div>
  </div>
</div>

Basic

Apply .placeholder to any inline element and give it a width via a grid column or sizing utility. The bar fills with the current text color at low opacity and shows a wait cursor.

<p class="w-100 mb-0 text-body-secondary" aria-hidden="true">
  <span class="placeholder col-6"></span>
</p>

Widths

Use grid column utilities (.col-*) or width utilities (.w-25, .w-50, .w-75, .w-100) to size each bar.

<div class="w-100 text-body-secondary" aria-hidden="true">
  <span class="placeholder col-6"></span>
  <span class="placeholder w-75"></span>
  <span class="placeholder" style="width: 25%;"></span>
</div>

Sizes

Three height modifiers sit alongside the default. .placeholder-xs and .placeholder-sm trim the bar for inline captions; .placeholder-lg bulks it up for headings.

<div class="w-100 text-body-secondary" aria-hidden="true">
  <span class="placeholder col-12 placeholder-lg"></span>
  <span class="placeholder col-12"></span>
  <span class="placeholder col-12 placeholder-sm"></span>
  <span class="placeholder col-12 placeholder-xs"></span>
</div>

Colors

Recolor with any .bg-* utility. The default picks up the current text color; wrap in .text-body-secondary for the muted gray used everywhere else on this page.

<div class="w-100" aria-hidden="true">
  <span class="placeholder col-12 text-body-secondary"></span>
  <span class="placeholder col-12 bg-primary"></span>
  <span class="placeholder col-12 bg-success"></span>
  <span class="placeholder col-12 bg-danger"></span>
  <span class="placeholder col-12 bg-warning"></span>
  <span class="placeholder col-12 bg-info"></span>
</div>

Animation

Wrap a group in .placeholder-glow for the pulsing opacity loop, or .placeholder-wave for a shimmer that sweeps across the bars.

<div class="d-flex flex-column gap-3 w-100 text-body-secondary" aria-hidden="true">
  <p class="placeholder-glow mb-0">
    <span class="placeholder col-12"></span>
    <span class="placeholder col-8"></span>
  </p>
  <p class="placeholder-wave mb-0">
    <span class="placeholder col-12"></span>
    <span class="placeholder col-8"></span>
  </p>
</div>

Buttons

Apply .placeholder to a .btn to render a disabled, unlabeled stand-in. Mirror the variant of the real button so the swap is seamless.

<div class="d-flex flex-wrap gap-2 w-100">
  <a class="btn btn-primary disabled placeholder col-3" aria-disabled="true"></a>
  <a class="btn btn-outline-secondary disabled placeholder col-2" aria-disabled="true"></a>
</div>
