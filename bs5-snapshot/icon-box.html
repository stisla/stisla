Icon box

A square, tinted container for a single icon.

Variants

Pair .icon-box with .icon-box-{state} for a tint in the matching brand color. The background sits at 15% and the icon holds the solid color.

<div class="d-flex flex-wrap gap-2">
  <span class="icon-box icon-box-primary"><i data-lucide="zap"></i></span>
  <span class="icon-box icon-box-secondary"><i data-lucide="settings"></i></span>
  <span class="icon-box icon-box-success"><i data-lucide="check"></i></span>
  <span class="icon-box icon-box-danger"><i data-lucide="alert-triangle"></i></span>
  <span class="icon-box icon-box-warning"><i data-lucide="clock"></i></span>
  <span class="icon-box icon-box-info"><i data-lucide="info"></i></span>
</div>

Shape

Stack Bootstrap's rounded utilities on top to switch shape without redefining anything. Use .rounded-circle for a circle or .rounded-0 for a hard square.

<div class="d-flex flex-wrap gap-2 align-items-center">
  <span class="icon-box icon-box-primary"><i data-lucide="bell"></i></span>
  <span class="icon-box icon-box-primary rounded-circle"><i data-lucide="bell"></i></span>
  <span class="icon-box icon-box-primary rounded-0"><i data-lucide="bell"></i></span>
</div>

Custom color

Override --bs-icon-box-bg and --bs-icon-box-color inline for one-off tints that sit outside the theme palette.

<div class="d-flex flex-wrap gap-2">
  <span class="icon-box" style="--bs-icon-box-bg: rgba(168, 85, 247, 0.15); --bs-icon-box-color: #a855f7;">
    <i data-lucide="sparkles"></i>
  </span>
  <span class="icon-box" style="--bs-icon-box-bg: rgba(236, 72, 153, 0.15); --bs-icon-box-color: #ec4899;">
    <i data-lucide="heart"></i>
  </span>
  <span class="icon-box" style="--bs-icon-box-bg: rgba(20, 184, 166, 0.15); --bs-icon-box-color: #14b8a6;">
    <i data-lucide="leaf"></i>
  </span>
</div>
