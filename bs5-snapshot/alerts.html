Alerts

A contextual feedback strip.

Basic

Use .alert on its own for a neutral message.

Some neutral message here. 
<div class="alert w-100">
  <i data-lucide="info"></i>
  Some neutral message here.
</div>

Variants

Add a state modifier like .alert-success or .alert-danger. Background and border tint to the state, the icon takes its full color, and body text stays neutral so the message reads at a glance.

Some neutral message here.
Payment failed. Check your card details.
Some useful information here.
Your changes have been saved successfully.
Some features may not work.
<div class="d-flex flex-column gap-2 w-100">
  <div class="alert">
    <i data-lucide="info"></i>
    <div>Some neutral message here.</div>
  </div>
  <div class="alert alert-danger">
    <i data-lucide="x-circle"></i>
    <div>Payment failed. Check your card details.</div>
  </div>
  <div class="alert alert-info">
    <i data-lucide="info"></i>
    <div>Some useful information here.</div>
  </div>
  <div class="alert alert-success">
    <i data-lucide="check-circle-2"></i>
    <div>Your changes have been saved successfully.</div>
  </div>
  <div class="alert alert-warning">
    <i data-lucide="triangle-alert"></i>
    <div>Some features may not work.</div>
  </div>
</div>

Without icon

There's no icon wrapper class. The leading icon is any direct <svg> or <i> child. Skip it and the row reflows around the text.

Heads up, your trial ends in 3 days.
Your changes have been saved successfully.
<div class="d-flex flex-column gap-2 w-100">
  <div class="alert alert-primary">Heads up, your trial ends in 3 days.</div>
  <div class="alert alert-success">Your changes have been saved successfully.</div>
</div>

With heading and description

Add .alert-description for a stacked layout. The heading sits above the description, the icon aligns with the heading, and the row gains a bit of breathing room.

Alert Title
Alert Description
<div class="alert w-100">
  <i data-lucide="info"></i>
  <div class="alert-heading">Alert Title</div>
  <div class="alert-description">Alert Description</div>
</div>

Same layout, stateful variant.

Heads up
A new version is available. Refresh to load it.
Unsaved changes
Leaving this page will discard your edits.
Payment failed
We couldn't charge your card. Update billing details and retry.
<div class="d-flex flex-column gap-2 w-100">
  <div class="alert alert-info">
    <i data-lucide="info"></i>
    <div class="alert-heading">Heads up</div>
    <div class="alert-description">A new version is available. Refresh to load it.</div>
  </div>
  <div class="alert alert-warning">
    <i data-lucide="triangle-alert"></i>
    <div class="alert-heading">Unsaved changes</div>
    <div class="alert-description">Leaving this page will discard your edits.</div>
  </div>
  <div class="alert alert-danger">
    <i data-lucide="x-circle"></i>
    <div class="alert-heading">Payment failed</div>
    <div class="alert-description">We couldn't charge your card. Update billing details and retry.</div>
  </div>
</div>

Without an icon the heading and description take the leading column.

No icon
The icon column collapses when no <svg> / <i> child is present.
<div class="alert w-100">
  <div class="alert-heading">No icon</div>
  <div class="alert-description">The icon column collapses when no <code>&lt;svg&gt;</code> / <code>&lt;i&gt;</code> child is present.</div>
</div>

Action slot

.alert-action is the trailing slot. It pushes to the right on single-line alerts and centers vertically when a description is present. Drop in a button, link, or close control.

Message deleted successfully.
<div class="alert w-100">
  <i data-lucide="info"></i>
  <div>Message deleted successfully.</div>
  <div class="alert-action">
    <button type="button" class="btn btn-secondary btn-sm">Undo</button>
  </div>
</div>

Multiple actions and stateful variants.

Your session is about to expire.
New version available
Reload to pick up the latest changes.
<div class="d-flex flex-column gap-2 w-100">
  <div class="alert alert-warning">
    <i data-lucide="triangle-alert"></i>
    <div>Your session is about to expire.</div>
    <div class="alert-action">
      <button type="button" class="btn btn-ghost-secondary btn-sm">Stay</button>
      <button type="button" class="btn btn-warning btn-sm">Extend</button>
    </div>
  </div>
  <div class="alert alert-info">
    <i data-lucide="info"></i>
    <div class="alert-heading">New version available</div>
    <div class="alert-description">Reload to pick up the latest changes.</div>
    <div class="alert-action">
      <button type="button" class="btn btn-info btn-sm">Reload</button>
    </div>
  </div>
</div>

Dismissible

Put a .btn-close with data-bsdismiss="alert" inside .alert-action. Bootstrap's dismiss JS still fires and the close sits in flow next to any other action. Skip the .alert-dismissible class. Its absolute positioning isn't needed here.

Your changes have been saved.
Couldn't connect
We lost contact with the server. Try again in a moment.
<div class="d-flex flex-column gap-2 w-100">
  <div class="alert alert-success fade show">
    <i data-lucide="check-circle-2"></i>
    <div>Your changes have been saved.</div>
    <div class="alert-action">
      <button type="button" class="btn-close" data-bs-dismiss="alert" aria-label="Close"></button>
    </div>
  </div>
  <div class="alert alert-danger fade show">
    <i data-lucide="x-circle"></i>
    <div class="alert-heading">Couldn't connect</div>
    <div class="alert-description">We lost contact with the server. Try again in a moment.</div>
    <div class="alert-action">
      <button type="button" class="btn-close" data-bs-dismiss="alert" aria-label="Close"></button>
    </div>
  </div>
</div>

Link color

Use .alert-link for inline links so they pick up the variant color without extra utility classes.

A simple info alert with an example link.
<div class="alert alert-info w-100">
  <i data-lucide="info"></i>
  <div>A simple info alert with <a href="#" class="alert-link">an example link</a>.</div>
</div>
